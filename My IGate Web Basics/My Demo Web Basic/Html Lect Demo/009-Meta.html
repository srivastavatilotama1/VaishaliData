The HTML meta Element Metadata is information about data. The tag provides metadata about the HTML document. Metadata will not be displayed on the page, but will be machine parsable. Meta elements are typically used to specify page description, keywords, author of the document, last modified, and other metadata. The tag always goes inside the head element. The metadata can be used by browsers (how to display content or reload page), search engines (keywords), or other web services. -------------------------------------------------------------------------------- Keywords for Search Engines Some search engines will use the name and content attributes of the meta element to index your pages.
